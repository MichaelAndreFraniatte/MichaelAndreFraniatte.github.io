Audio for Desktop

Having Audio with Visualization on Your Desktop Screen

The program is for display and listen audio as overlay for your desktop. Let's see...

Considerations About the Program

I've been inspired by this, but take care it modify your OS. You can locate the overlay everywhere on your desktop, it will remember the location after restarting it. Double click on the notify icon to activate the overlay, and drag and drop the overlay with the border on top, then click anywhere else on your desktop to deactivate it. You can close it with right click on the notify icon. You can also set the overlay to start automatically at your PC launch, or unset it, but you need to start the overlay with administrator privilege. You shall tweak the HTML file under assets folder from where is the program to add new audio files. Check my program named rmomo for tweak the HTML file with auto creation of audio tags in a file to copy/paste.
