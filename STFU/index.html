Spotify Tracks For You

Easy Tool to Capture Audio from Sound Card on Spotify

The program is made for capture audio sound from sound card for recording perfectly songs over Spotify with dedicated browser. The files recorded are in MP3 format. Let's see...

Considerations About the Program

It stoping capture after the end of the current track, and restart before the next track, recording each time a song in a new file. It automatically records the file with the good name, and with tags like title and artist of song. It's the features of rmomo and AlbumCapture but in a better shape for record easy songs and add it directly to DesktopAudio or DesktopOverlay. You just need to trigger twice the record for starting but after it trigger alone the record for each song you listen with right flow. If the name of title is too long it need to use rmomo to change title's tag. You having a far better audio sound listening with the mp3 recorded. For using program, it need a key file for authenticate locally in order to be able to start properly, ask me to obtain this key file (it depend of your computer, not of the program).
