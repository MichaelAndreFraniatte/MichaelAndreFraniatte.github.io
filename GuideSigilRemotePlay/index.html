SIGIL Remote Play

Configuring Console Remote Play

To use any controller on console you need SIGIL for handle controller. For remote play you need to enable it under your console. To use remote play, you need a PC with XboxApp or Greenlight or XStreaming. This last program can be used with console display in 720p, otherwise only 1080p resolution is working. The remote play program you are using must be in front window but you can use an overlay in Topmost, enabling fullscreen with keyboard key F11. It need to enable a Xbox controller on your PC. Using a Virtual Xbox Controller

You can play with any controller using SIGIL simulating a virtual Xbox controller, but you need to hide other controller from remote play program. The driver Devreorder is not working to hide controllers but HidHide program is working. You need to add remote play programs in HidHide program. It's the same steps as with Steam, but with remote play programs. If you want more workaround about HidHide, check my guide for it.

All It Working Now on Consoles

Following this guide lead you to make working any controller as a virtual Xbox controller on console. You can switch to screen of your console display for best gaming experience. Games in 720p are working better without any lag, and it secure in time your console. Also your PC used just for remote play feel better. It don't need a good PC, even a good network connection.
